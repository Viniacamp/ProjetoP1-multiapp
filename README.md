# ProjetoP1-multiapp

Nome: Vinicius Soares de Almeida

RA: 2171392411026

O projeto consiste em um compilado de mini-apps reunidos em um aplicativo só, demonstrando vários dos conceitos aprendidos durante as aulas:

- Projeto MegaSena: Geração de 6 números aleatórios simulando apostas da Mega-Sena.

- Projeto IMC (Índice de Massa Corporal): Cálculo do IMC com base em peso e altura, exibindo a classificação correspondente.

- Questionário Felicidade: Um sistema de avaliação baseado em hábitos diários, que calcula um índice de felicidade utilizando pontuação ponderada**
